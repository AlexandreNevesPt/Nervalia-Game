Welcome to the Nervalia-Game wiki!

How to Install and Start Playing Nervalia: To start playing the game, download the ZIP file. After downloading, extract its contents. Launch the project by opening the script "menu_mod.py". If Visual Studio Code or Visual Studio shows an error related to PIL, run the following command in the terminal or console: pip install pillow

Game Story: You are a Mushroom Knight on a wild adventure to find your lost princess. During your journey, you will have to make choices, and these choices will determine your path.
